Wystarczy pobrać plik .exe (wrzuciłem też kod źródłowy jako .py do podglądu). 
Po zainstalowaniu aplikacji należy wkleić link do profilu na ogladajanime oraz wybrać miesjce, w którym zostaną zapisane wyniki.
Po ukończeniu kopiowania listy w wybranym miejscu pojawią się 3 pliki: .xlm (skopiowana lista), .txt (nieznalezione tytuły zostaną w nim zapisane) oraz .json (zwolnienie pamięci np. dla następnego profilu)
Otrzymany plik .xlm należy wrzucić na stronę: https://myanimelist.net/import.php wybierając MyAnimeList Import
